# CHUCK70XHI.2
this is the code and model for my interactive interface embedded into a Converse Chuck with visualises the gait, interfacing your expression/mood. i used tiny motion trainer to train the IMU data to reccognise two different gaits, Stressed and Calm: https://experiments.withgoogle.com/tiny-motion-trainer/view/capture 
# download and open files in arduino to run 
you will need a: 
1. 3.7 volt lipo battery
2. Generic USB Type C 18650 TP4056 Lithium Battery charger
3. Arduino Nano 33BLE SENSE
4. 1.3 inch OLED display
   
# make sure to have the u8g2 library downloaded for graphics
